# mini_forum
一个前后端分离的轻量级的论坛，功能如：discuz 仿制诸如推特、百度贴吧等功能。支持👨‍🔬用户注册、🕔主题切换、🌏国际化、富文本编辑和发布帖子、支持用户自定义组件、设置tag与板块等功能。同时管理员可以通过后台快速设置论坛内容、禁言用户等。
